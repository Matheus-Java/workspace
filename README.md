# Configurando Ambiente de Desenvolvimento

##### Programas Necessários:
* [Chocolatey](https://chocolatey.org/install)
* [Composer](https://getcomposer.org/download/)
* [Fira Code (Fonte)](https://github.com/tonsky/FiraCode)
* [Git](https://git-scm.com/download/win)
* [Insomnia](https://insomnia.rest/download)
* [Laragon](https://laragon.org/download/index.html)
* [Node.js](https://nodejs.org/en/download/)
* [Visual Studio Code (Editor de código)](https://code.visualstudio.com/download)

##### Configurando Windows: 
Instale o **Windows Terminal** [click aqui](https://www.microsoft.com/pt-br/p/windows-terminal/9n0dx20hk701?activetab=pivot:overviewtab).

###### Personalização do Cursor:
Altere o cursor do mouse do windows baixando os arquivos na pasta **cursor-windows**.

##### Configurando o Editor de Código:
Abra o **Settings.json** do VSCode e copie e cole as settings do arquivo no repositório **Settings.json**.
